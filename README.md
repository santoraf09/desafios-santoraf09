## Descrição dos Desafios

### **1. Cálculo da variável SOMA**
Um programa que calcula o valor final de uma variável SOMA com base em incrementos definidos por um loop.

- **Entrada:** Nenhuma entrada é necessária.
- **Saída:** O valor final da variável SOMA.
- **Arquivo:** `questao1.py`

---

### **2. Sequência de Fibonacci**
Um programa que verifica se um número informado pertence à sequência de Fibonacci.

- **Entrada:** Um número inteiro informado pelo usuário.
- **Saída:** Uma mensagem indicando se o número pertence ou não à sequência.
- **Arquivo:** `questao2.py`

---

### **3. Análise de Faturamento Diário**
Um programa que processa um arquivo JSON contendo dados de faturamento diário e retorna:
- O menor valor de faturamento.
- O maior valor de faturamento.
- O número de dias em que o faturamento diário foi superior à média mensal.

- **Entrada:** Arquivo JSON com os dados de faturamento.
- **Saída:** Valores calculados com base nos dados do JSON.
- **Arquivo:** `questao3.py`
- **Fonte de dados:** `faturamento.json`

---

### **4. Percentual de Faturamento por Estado**
Um programa que calcula o percentual de representação do faturamento mensal de cada estado no total mensal.

- **Entrada:** Valores de faturamento por estado (fixos no código).
- **Saída:** Percentual de contribuição de cada estado.
- **Arquivo:** `questao4.py`

---

### **5. Inversão de String**
Um programa que inverte os caracteres de uma string informada pelo usuário, sem usar funções prontas como `reverse`.

- **Entrada:** Uma string digitada pelo usuário.
- **Saída:** A string invertida.
- **Arquivo:** `questao5.py`

---

## Pré-requisitos

Para executar os scripts, é necessário ter:
- **Python 3.8+** instalado.
- Um terminal ou IDE configurada para rodar arquivos `.py`.
- O arquivo JSON `faturamento.json` na mesma pasta do script para a **Questão 3**.
